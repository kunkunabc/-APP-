# -工作流提效项目一款基于a~i开发的商业（电商）领域数据报表的双周回流分析自动处理器app项目-

本项目未仅作为实习生面向百度商业（电商）投放运营的日常各种数据报表分析和处理效率较低易出错自主设计的提效小工具，其中示例的所有数据已进行数据脱敏或者脏数据混合处理，仅用于交流学习，禁止商业使用！

实习用于电商数据自动广告投放数据针对回流数据及其投放数据结果的报表分析和处理等过程中，针对周度子表级的双周数据的各项ROI、CPC等投放指标等自动分析和计算，避免了人工数据处理任务过程中效率低下、易犯错等问题。
